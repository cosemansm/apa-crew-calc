01

Brand Identity

Crew Dock is a payroll and scheduling tool built for UK film & TV crew. It calculates wages, overtime, and turnaround costs against APA recommended terms — precisely and reliably.

The Logo
App icon.png
On yellow bg
White on dark
The lighthouse represents guidance, navigation, and safety — concepts that resonate with the chaos of a film shoot. The pixel aesthetic signals precision and craft without being cold.
Logo Rendering Rules
  • → Always display inside a #FFD528 container
  • → Use mix-blend-mode: multiply on light backgrounds
  • → Canonical: 40×40px with rounded-2xl
  • ✕ Never on dark bg without yellow container
  • ✕ Never stretch, recolour, or add drop shadow

07

Motion & Animation

All transitions use 200ms. Fast enough to feel snappy. Slow enough to register. The brand is precise and calm — no bounce, no spring.

Button hover
transition-all duration-200 · shadow + brightness
Button press
active:scale-[0.98] · instant
Card hover
transition-shadow duration-200
Input focus
transition-colors duration-200
Dialog open/close
zoom-in-95 / zoom-out-95 + fade-in-0 / fade-out-0
Popover / select
Same as dialog + slide-in from edge
Switch toggle
transition-transform · thumb translate-x-0 → translate-x-5
Nav item
transition-all duration-200
Do
  • Animate opacity, transform, shadow
  • 200ms as the default duration
  • Transitions on interaction only
  • 300ms max for full-page transitions
Don't
  • Don't animate width, height, or padding
  • Don't use bounce or spring easing
  • Don't animate on page load

Focus & Accessibility

Every interactive element uses the same yellow focus ring. It is highly visible against both white surfaces and the dark sidebar.

/* Applied universally */
focus-visible:ring-2
focus-visible:ring-ring    /* #FFD528 */
focus-visible:ring-offset-2
Contrast Ratios

Sample Text
#1F1F21 on #FFD528
9:1 — AAA ✓
Sample Text
#8A8A8A on #FFFFFF
3.6:1 — Use sparingly
Do
  • Use focus-visible not focus
  • Pair disabled:opacity-50 with pointer-events-none
  • Always explain why an action is unavailable
Don't
  • Never remove the focus ring
  • Never use outline: none without a replacement
  • Never use muted text for critical information
09

Tone of Voice

Crew Dock serves experienced UK film & TV crew. Tight budgets, long days, complex APA rules. The tone must match: direct, precise, calm.

TraitMeaning
DirectNo waffle. Say what it does.
PreciseUse exact numbers and terms. "11 hours" not "over ten hours".
ProfessionalNo emojis, no exclamation marks in the UI.
KnowledgeableUse industry language: TOC, turnaround, basic rate, call fee.
CalmErrors are informative, not alarming. Warnings are factual, not dramatic.
Copy Examples
Avoid
Oops! Something went wrong. Please try again later!
Use
Rate is required. Enter a daily or hourly rate to continue.
Avoid
Click here to add a powerful new job to your amazing dashboard
Use
No jobs yet. Add a job to start calculating pay.
Avoid
You've worked over ten hours today — great work! 🎉
Use
11h 30m basic time. Overtime starts at 11h 00m.
10

Design Don'ts

A quick reference. When in doubt, come back to this list first.

✕
Introduce a second brand accent colour
✕
Use pure #000000 or pure #FFFFFF as background
✕
Put two primary buttons on the same screen
✕
Use italic text anywhere
✕
Use text-4xl or larger inside the app
✕
Set financial figures in a proportional font
✕
Use bg-primary/50 — use secondary or ghost variant
✕
Remove or replace the focus ring
✕
Animate layout properties (width, height, padding)
✕
Use bounce or spring easing
✕
Nest cards inside cards
✕
Display the logo without its yellow container
✕
Add decorative gradients inside cards or components
✕
Use green, purple, or red outside of status badges
✕
Use emojis, exclamation marks, or marketing language in app copy
✕
Use arbitrary spacing values — stick to the 4px Tailwind grid
